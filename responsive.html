% vs vw vs max-width (and Media Query)

Using % Width

width:(a percentage);

Behavior: Green box is 50% of its parent container. Fully depends on the gray box size.

Using vw Width

width: a number and "vw";

Behavior: Blue box is 50% of the full browser width, ignoring the gray parent’s width.

Using % + max-width (vw)

width: 70%; max-width: 60vw;

Behavior: Purple box grows with its parent (70%) but never exceeds 60% of the viewport width. On screens under 800px, it turns pink and expands to 90%.

Responsive Text Examples

These examples show how text can scale using %, vw, or clamp().

Parent container font-size: 16px

Text using font-size: a percentage

This text is 120% of the parent’s 16px font size = 19.2px. If we change the parent font-size (try 20px or 14px), the text adjusts automatically.

Text using font-size: 3vw

This text scales directly with the browser’s width. Resize the window and watch it grow and shrink — useful for headlines, but can get tiny on phones.

Text using font-size: clamp(1rem, 2vw, 2rem)

This balances flexibility and readability. The text grows with the screen but never smaller than 1rem or larger than 2rem. On smaller screens (under 600px), it turns pink.
